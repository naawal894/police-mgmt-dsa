# 👮 Police Management System (DSA)

A C++ Console Application designed to manage law enforcement records efficiently. This project focuses on **File Handling** and **Data Structures** to maintain records for prisoners, officers, and prison cells.

## 🧠 Data Structures & Logic
* **File Handling:** Uses `.txt` files (`prisoners.txt`, `officers.txt`) as a flat-file database to persist data.
* **Searching:** Implemented logic to lookup specific records by ID or Name.
* **Record Management:** Supports full CRUD (Create, Read, Update, Delete) operations for police data.

## ✨ Features
* **Prisoner Tracking:** Manage names, IDs, and cell assignments.
* **Officer Database:** Keep track of personnel on duty.
* **Cell Management:** Monitor prison cell capacity and status.
* **Persistent Storage:** Data is saved automatically to text files.

## 🚀 How to Run
1. Open the folder in **Visual Studio**.
2. Click on `ConsoleApplication1.sln`.
3. Build and Run (`Ctrl + F5`).
4. Make sure the `.txt` files are in the same directory as the executable.
